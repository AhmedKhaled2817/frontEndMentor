# Contact Form

This is a Contact Form project built to enhance front-end skills by creating interactive and responsive web forms.  
The project is made using *HTML5, **CSS3, and **JavaScript*, focusing on clean UI and UX.

## Features
- Clean and modern UI
- Mobile responsive design
- Basic JavaScript form validation

## Technologies Used
- HTML5
- CSS3
- Bootstrap5
- JavaScript
